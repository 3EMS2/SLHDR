# SLHDR WRAPPER

This repository is C wrapper to use Interdigital SLHDR library.

## Build Step
`g++` in version 7 or higher is required to build this wrapper
```
./configure --sse_folder=path/to/folder/sse --avx2_folder=path/to/folder/avx2
make
```
